# OCaml Allegro 5

This is an OCaml binding for the [Allegro 5](https://liballeg.org/) C library.

You shall install it with:
```sh
opam install allegro5
```

## Dune commands

```sh
dune build
dune build @doc
dune test
dune install
dune uninstall
```

You will find the docs at `_build/default/_doc/_html/index.html`.
